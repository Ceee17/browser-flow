## Architecture Decisions

- JSON workflows are the primary artifact. Edits create new versions; existing versions are immutable.
- Steps are a discriminated union on `type` (Zod `z.discriminatedUnion`).
- Selectors are modeled as a set of candidates with `confidence`, enabling multi-strategy fallback.
- Runner is outbound-only; control plane never executes browser automation.
- Stack defaults: TypeScript strict, Zod validation, Playwright, NestJS, Next.js, PostgreSQL, Redis + BullMQ.
